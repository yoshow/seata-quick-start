## seata-quick-start ##

基于 Spring Cloud Alibaba 的 Seata 快速入门示例

示例环境相关版本
| 名称                  | 版本    |
| :--                   | :--     |
| Spring Cloud Alibaba  | 2.2.3   |
| Seata                 | 1.3.0   |

如何编译此项目?
```
mvn install
```

环境要求 Java 1.8


测试地址
http://localhost:18001/trans/service1/method1-at
